很开心呢，昨天终于自主的买了代理、自己部属，我可以自己使用Google翻墙了~哈哈，今天又成功搭建了自己的blog（以前虽然也使用了，但是别人帮我搞的环境）~话不多说了，今天我就详细的整理下搭建博客的过程，希望可以帮到小白白们~~~

前期准备

为什么要搭建一个独立博客

独立的才是自己的

小白进入门槛

1、非常折腾，需要耐心
2、需要一定的学习能力和钻研精神
3、懂一些Markdown语法和网页知识，不懂请参见第一、二条

为什么要选择github pages

1、github pages有300M免费空间，资料自己管理，保存可靠
2、学着用github，享受github的便利，上面有很多大牛，眼界会开阔很多
3、顺便看看github工作原理，最好的团队协作流程（提一句，和瀑布配合使用效果更佳哦）
4、github是趋势
5、逼格呀

github pages是什么

github pages本用于介绍托管在GitHub的项目，由于他的空间免费稳定，用来做搭建一个博客再好不过了

添加SSH Key到GitHub

在本机设置SSH Key之后，需要添加到GitHub上，以完成SSH链接的设置
1、打开本地C:\Users\Hasee.ssh文件，此文件里面内容为刚才生成人密钥。如果看不到这个文件，你需要设置显示隐藏文件。准确的复制这个文件的内容，才能保证设置的成功
2、登陆github系统，点击右上角的 Account Settings—>SSH Public keys —> add another public keys
3、把你本地生成的密钥复制到里面（key文本框中），点击 add key 就ok了
